Get Started

Build and run MyOs in minutes with these simple steps

Requirements

  • x86_64-elf-gcc, x86_64-elf-ld - Cross-compiler toolchain
  • nasm (Netwide Assembler) - Assembly language compiler
  • grub-mkrescue - Bootable ISO creator
  • qemu-system-x86_64 - Emulator for testing

Build & Run

‚åò 1. Clone & Build ISO
git clone https://github.com/your-repo/myos.git && cd myos && make

Downloads source and creates build/MyOs.iso

‚åò 2. Run in QEMU
make run

Launches with 8MB auto-created disk image

‚åò 3. Clean build artifacts
make clean

Removes build directory, ISO and disk image
